MobSF이란?

Mobile Security Framework (MobSF) is a security research platform for mobile applications in Android, iOS and Windows Mobile. MobSF can be used for a variety of use cases such as mobile application security, penetration testing, malware analysis, and privacy analysis. The Static Analyzer supports popular mobile app binaries like APK, IPA, APPX and source code. Meanwhile, the Dynamic Analyzer supports both Android and iOS applications and offers a platform for interactive instrumented testing, runtime data and network traffic analysis. MobSF seamlessly integrates with your DevSecOps or CI/CD pipeline, facilitated by REST APIs and CLI tools, enhancing your security workflow with ease.
MobSF(Mobile Security Framework) 는 Android, iOS 및 Windows Mobile의 모바일 애플리케이션을 위한 보안 연구 플랫폼입니다. MobSF는 모바일 애플리케이션 보안, 침투 테스트, 멀웨어 분석 및 개인 정보 보호 분석과 같은 다양한 사용 사례에 사용될 수 있습니다. Static Analyzer는 APK, IPA, APPX 및 소스 코드와 같은 인기 있는 모바일 앱 바이너리를 지원합니다. 한편 Dynamic Analyzer는 Android 및 iOS 애플리케이션을 모두 지원하며 대화형 계측 테스트, 런타임 데이터 및 네트워크 트래픽 분석을 위한 플랫폼을 제공합니다. MobSF는 REST API 및 CLI 도구에 의해 촉진되는 DevSecops 또는 CI/CD 파이프라인과 원활하게 통합되어 보안 워크플로우를 쉽게 향상시킵니다. - MobSF/Mobile-Security-Framework-MobSF

공식 홈페이지에서는 이렇게 설명하고 있습니다. MobSF(Mobile Security Framework)는 Android, iOS 및 Windows Mobile의 보마일 애플리케이션을 자동으로 분석하는 플랫폼입니다. 정적/동적 분석을 지원하고 있습니다.

Installation on Docker

예전에는 소스코드를 직접 다운로드 하여 환경을 구성하여 구동하였지만 현재는 Docker를 이용하여 손쉽게 구동할 수 있습니다.

$ docker pull opensecurity/mobile-security-framework-mobsf:latest
$ docker run -it --rm -p 8000:8000 opensecurity/mobile-security-framework-mobsf:latest

Android Emulator 설치 및 설정

동적 분석기를 사용하기 위해서는 Android Emulator가 설치가 되어 있어야지만 사용할 수 있는 기능입니다.
동적 분석기를 사용할 때에는 Docker 실행 명령어를 추가하여 실행 해야합니다.

$ 

$ emulator -list-avds
Pixel_7_Pro_API_28
Pixel_7_Pro_API_33
$ emulator -avd Pixel_7_Pro_API_28 -writable-system -no-snapshot

주의사항이 있습니다. 에뮬레이터 설치하면서 API 28 버전 이상에서는 지원을 하지 않습니다.
최신 버전의 API를 사용할 경우 MobSF를 사용할 수 없습니다.

Only Android images upto API 28 is supported! Newer Android AVDs does not offer a writable /system and hence cannot work with MobSF.
API 28까지의 Android 이미지만 지원합니다! 더 새로운 Android AVD는 쓰기 가능/시스템을 제공하지 않으므로 MobSF와 함께 작업할 수 없습니다.
